Jenkins CI/CD Pipeline

Modern Java Deployment Architecture

1. Maven

Compiles the Java code, runs unit tests, and builds the executable .jar artifact.

2. SonarQube

Scans the code for bugs, vulnerabilities, and code smells to ensure Quality Gate passes.

3. Docker

Packages the Java app into a lightweight container image and pushes it to a registry.

4. Helm

Updates the Kubernetes deployment charts with the newly generated Docker image tag.

5. Argo CD

Detects changes in the Git repository (GitOps) and automatically syncs the new state.

6. Kubernetes

Pulls the new image and orchestrates the running of the updated application pods.
